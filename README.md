## Skills Inventory System
### A skills inventory system shows a comprehensive list of all the experiences, professional skills, and educational qualifications of employees in this organization. It also aims to monitor one's current skill portfolio and reveal areas for growth and development. Additionally, it can also be utilized for recruitment and promotion.
